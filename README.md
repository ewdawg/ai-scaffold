# AI Agent Scaffold

A reusable template repository to instantly kickstart AI-assisted coding projects with best-in-class agent rules, behavioral constraints, and tool discovery instructions.

## How to use this scaffold

1. Clone or copy this repository into your new project folder:
   ```bash
   git clone https://github.com/yourusername/ai-scaffold.git new-project
   cd new-project
   rm -rf .git  # Remove the scaffold's git history
   git init     # Start fresh
   ```
2. Open `AGENTS.md`.
3. Search for the `[TODO]` blocks and fill in the details about your new project's Tech Stack, Architecture, and Safety Rules.
4. Point your favorite AI coding agents (Gemini, Claude, Cline) to `AGENTS.md` so they all share a single source of truth!
